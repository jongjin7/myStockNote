# UI 구현 가이드 (Concise Spec)

## 1. 디자인 시스템 (Core)

- **테마**: 프리미엄 다크 모드 (배경: `slate-950`, 카드: `slate-900/40`)
- **포인트 컬러**: 블루(`primary-500`), 그린(`success`), 레드(`danger`)
- **폰트**: `Pretendard` 단일화 (숫자는 `tabular-nums` 적용)
- **효과**: 1px 보더(`white/5`), 미세한 블러(`backdrop-blur-sm`)

## 2. 레이아웃 & 네비게이션

- **PC**: 고정 사이드바 (좌) + 스크롤 콘텐츠 (우)
- **Mobile**: 하단 탭 바 (홈, 계좌, 관심, 설정)
- **전환**: 모든 페이지 진입 시 Slide-up + Fade (300ms)

## 3. 화면별 핵심 명세

### 3.1 대시보드
- **Hero**: 총 자산 가치 상단 배치 (가장 큰 폰트)
- **Stats**: 보유 종목/노트 수 카드를 그리드 형태로 배치

### 3.2 계좌 관리
- **카드**: 증권사명, 예수금, 종목 수 표시
- **예수금**: 원화 정수 단위, 3자리마다 콤마 표시

### 3.3 계좌별 종목 리스트
- **Summary**: 계좌 총 자산, 투자금, 누적 수익 요약 스트립
- **List**: 종목별 실시간 수익률 배지(Badge) 포함

### 3.4 종목 상세
- **Evaluation**: 현재가(Yahoo API), 평가금, 수익금(금액/%) 카드
- **Timeline**: 투자 노트를 최신순으로 수직 배치 (타입별 보더 컬러 구분)

## 4. 디자인 제약사항

- **가독성**: 최소 텍스트 크기 `text-sm`, 주요 수치는 `text-base` 이상
- **인터랙션**: 모든 인터랙티브 요소에 호버(`scale-105` 등) 및 액티브 효과 필수
- **빈 상태**: 데이터 없을 시 가이드 아이콘 및 문구 노출

## 5. 기술적 구현

- **API**: Yahoo Finance + AllOrigins Proxy (CORS 회피)
- **성능**: GPU 가속 애니메이션(transform, opacity) 사용
